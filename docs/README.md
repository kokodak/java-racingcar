## Domain

### Car

- [x] 자동차는 이름을 가진다.
- [x] 자동차는 위치를 가진다.

- [x] 경주할 자동차의 정보를 가진다.
- [x] 이름으로 자동차를 조회할 수 있다.

- [ ] 진행 상태를 출력할 수 있다.
